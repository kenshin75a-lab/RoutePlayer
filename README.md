# 🏍 RoutePlayer

> Ride Before You Ride

오토바이 라이더를 위한
Street View 기반 라이딩 시뮬레이터

---

## 목표

라이딩을 출발하기 전에

- 실제 도로
- 커브
- 교차로
- 도로폭
- 풍경

을 미리 확인할 수 있는 프로그램.

---

## 현재 기능

✅ Google Maps

✅ OpenRouteService 경로 생성

✅ 지도에 경로 표시

---

## 개발 예정

- Google Places 주소검색
- Street View 재생
- GPX 가져오기
- 추천 라이딩 코스
- 라이딩 플레이어
